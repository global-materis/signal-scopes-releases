# SignalScope — Releases

Instaladores oficiales de **SignalScope**, el monitor de enlaces inalámbricos
para operadores WISP (Ubiquiti airMAX).

- Descarga la última versión desde
  [Releases](https://github.com/getwisnee/signal-scopes-releases/releases/latest)
  (`SignalScope_x.y.z_x64-setup.exe`).
- La aplicación instalada avisa sola cuando hay una versión nueva.
